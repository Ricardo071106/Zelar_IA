# 📊 Guia do Sistema de Analytics - Zelar

## 🚀 Como Acessar

### **Interface Web (Recomendado)**
```
https://zelar-ia.onrender.com/analytics
```

### **API JSON**
```
https://zelar-ia.onrender.com/api/analytics
```

## 📈 O que você vai ver

### **📊 Resumo Geral**
- **Total de Mensagens**: Todas as interações
- **Usuários Únicos**: Pessoas diferentes que usaram
- **WhatsApp**: Mensagens via WhatsApp
- **Telegram**: Mensagens via Telegram

### **📈 Top Eventos**
- Eventos mais agendados (Jantar, Reunião, Prova, etc.)
- Contagem de cada tipo

### **💬 Top Prompts**
- Frases mais usadas pelos usuários
- Ex: "jantar hoje às", "reunião amanhã"

### **🕐 Uso por Hora**
- Picos de atividade durante o dia
- Horários mais movimentados

### **📅 Últimos 7 Dias**
- Crescimento diário
- Tendências de uso

### **🆕 Atividade Recente**
- Últimas 10 mensagens
- Usuário, plataforma, mensagem, horário

## 🔄 Atualização Automática

- **Interface Web**: Atualiza a cada 30 segundos
- **Botão "🔄 Atualizar"**: Atualização manual

## 📱 Como Funciona

1. **Rastreamento Automático**: Toda mensagem é registrada
2. **Privacidade**: IDs são anonimizados
3. **Tempo Real**: Dados atualizados instantaneamente
4. **Histórico**: Mantém últimos 1000 logs

## 🎯 Exemplos de Uso

### **Para Negócio**
- Ver quais eventos são mais populares
- Identificar horários de pico
- Acompanhar crescimento de usuários

### **Para Desenvolvimento**
- Testar funcionalidades
- Verificar se bots estão funcionando
- Monitorar erros

## 🔧 Dados Coletados

- ✅ **Plataforma** (WhatsApp/Telegram)
- ✅ **Tipo de Evento** (Jantar, Reunião, etc.)
- ✅ **Horário** (Hora e dia)
- ✅ **Usuário** (Anonimizado)
- ✅ **Mensagem** (Primeiras palavras)

## 🚫 Dados NÃO Coletados

- ❌ Números de telefone
- ❌ Nomes reais
- ❌ Mensagens completas
- ❌ Dados pessoais

## 📊 Interpretando os Dados

### **Crescimento Saudável**
- Aumento gradual de usuários únicos
- Uso consistente em diferentes horários
- Variedade de tipos de eventos

### **Pontos de Atenção**
- Poucos usuários únicos
- Uso concentrado em um horário
- Poucos tipos de eventos
